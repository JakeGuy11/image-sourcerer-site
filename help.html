How To Use

   Whenever you see an image on supported sites, a red download icon will appear somewhere near it - it may be on a post the image is from, it may be in a nearby convienient spot, it may even be on the image itself. When you see one of these buttons around an image you would like to download, click it and follow the pop-up's instructions. Sometimes, there is only one download button for multiple images. When this is the case, you will be prompted for which image to download. When prompted for where to save the image, exclude the file extension. Non-latin characters are allowed. Images will be saved in [your downloads folder]/Image-Sourcerer/[path that you enter]. Adding a / to the path will save it into a subfolder.

   If you have followed these instructions but are still encountering a problem, do the following; Make sure you're on a supported site. The extension will never work if the site is not supported! Then, check the Known Bugs section. Some bugs are known about and are being worked on. If you still haven't found an answer, please navigate to the Report a Bug section below to let us know about it!

Supported Sites

   Image Sourcerer is an extension that allows you to download images from supported sites. If the site is not explicitly supported by the extension, you will not be able to use it. The following is a list of currently supported sites and their statuses:
  • Twitter
    • Support is complete, no known bugs
  • Pixiv
    • Support is nearing completion, 3 known bugs
  • Danbooru
    • Support is complete, no known bugs
  • Reddit
    • Support is minimal - do not use
You can read more about any mentioned bugs in the Known Bugs section below. If you would like to see a site get support, read the Request a Site section below. If you would like an automatic form for site requests, consider donating in order to upgrade the server!

Transferring Images

   Image Sourcerer stores data inside the images you download by appending a base-64 string on the end of the image. This is fairly easy to accomplish, but because it's not nessisary to the actual image data, some tools will remove it when you open the image. In general, any tool or site that can change the image's contents will remove it. Sites like Twitter and imgur also downscale images to save storage, so the data will be lost if you upload it to those. Raw transfers will not loose the data, for example sending it over Discord or SMS. Some email services will remove it, so be cautious.

Request A Site

   As mentioned above, only supported sites let you download images from them. Programming support for a site can take anywhere from a week to multiple months depending on how it is designed. Because of this, it may take a while to add support for a site you request, so please be patient.

   There are a few ways to request support for a site. First, and most effective, is to join the Discord and send your request to #site-request. Alternatively, you can email us at image.sourcerer.extension@gmail.com.

   If you would like an official form for site requests, please consider donating so we can afford a server that supports it! You can find more details in the Contribute page.
