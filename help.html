How To Use

   Whenever you see an image on supported sites, a red download icon will appear somewhere near it - it may be on a post the image is from, it may be in a nearby convienient spot, it may even be on the image itself. When you see one of these buttons around an image you would like to download, click it and follow the pop-up's instructions. Sometimes, there is only one download button for multiple images. When this is the case, you will be prompted for which image to download.

   If you have followed these instructions but are still encountering a problem, do the following; Make sure you're on a supported site. The extension will never work if the site is not supported! Then, check the Known Bugs section. Some bugs are known about and are being worked on. If you still haven't found an answer, please navigate to the Report a Bug section below to let us know about it!

Supported Sites

   Image Sourcerer is an extension that allows you to download images from supported sites. If the site is not explicitly supported by the extension, you will not be able to use it. The following is a list of currently supported sites and their statuses:
  • Twitter
    • Support is complete, no known bugs
  • Pixiv
    • Support is nearing completion, 3 known bugs
  • Danbooru
    • Support is complete, no known bugs
  • Reddit
    • Support is minimal - do not use
You can read more about any mentioned bugs in the Known Bugs section below. If you would like to see a site get support, read the Request a Site section below. If you would like an automatic form for site requests, consider donating in order to upgrade the server!

Known Bugs

   Below is a list of known bugs. The ■ green entries are currently being resolved by a developer, the ■ yellow entries are in the backlog and will be resolved in the near future, the ■ red entries are known and in the backlog, but will likely not be resolved at any time in the near future, and ■ blue entries are not bugs. Read their description for further information. This list is subject to change, so be sure to check it often.
  • Pixiv
    • Download button does not load on some sites. Temporary work-around: reload the page until it shows.
    • On pages with more than 5 or 6 images, the download functionality is disabled. There is no work-around currently.
    • Download times on all Pixiv images are very long. This is because a proxy server is needed for downloads. Consider donating to decrease the download times! Note that you must wait until the download is complete to close the tab, or else the image will not download!
