Installation Instructions

   These instructions assume you've already downloaded the version you want to install. The beta versions and working versions are linked below. Follow these instructions regardless of which version you're installing:
  1. Extract the zip into a safe place. A good place is somewhere like your Documents folder.
  2. Navigate to [browser]://extensions. This will take you to the extension page. [browser] should be replaced with the name of your browser, ex. chrome://extensions, brave://extensions, etc.
  3. Toggle Developer Mode to on. There will be a switch in one of the corners labeled "developer mode". Toggling it on will allow you to load extensions from a folder.
  4. Click "Load unpacked" and navigate to the folder where you extracted Image Sourcerer. Select it.
  5. Start using Image Sourcerer! The installation is complete! If you overwrite the local files with a newer version, you will need to hit the reload button on the Image Sourcerer panel in the extension page in order for the changes to take effect.
